# tg-bot-ai_assistant

Небольшой Telegram-бот на Python с логикой “AI assistant” (по структуре проекта: `gpt.py` + `handlers.py`).

## Что внутри

- `main.py` — точка входа (entrypoint - точка запуска)
- `handlers.py` — обработчики сообщений/команд
- `gpt.py` — логика, связанная с AI-ответами
- `parser.py` — парсер/утилиты для разбора данных
- `custom_data.txt` — дополнительные данные, которые использует проект
- `requirements.txt` — зависимости
